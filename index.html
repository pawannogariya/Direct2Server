Direct2Server

Direct2Server is a tiny JS library for MVC pattern applications, which allows you to directly bind server methods with DOM element events

Download .zip Download .tar.gz View on GitHub

What it is?

It is an abstraction over jquery ajax function that allows you to call any server method just like javascript methods, asynchronously on DOM events without writing a single piece of code for ajax requests.

So if you have a controller method like this

public class HomeController : Controller
{
    public void HiServer()
    {
            
    }
}

You can call it directly by just using attribute onAjaxServerClick like this

<input type="button" onAjaxServerClick="HomeController.HiServer" value="Hi Server!" />

You can call server methods for any DOM event by just putting phrase AjaxServer while calling it

So for e.g.

OnAjaxServerChange for OnChange

OnAjaxServerKeyUp for OnKeyUp

etc.

For all this to work, you just have to include Direct2Server js file along with any version of jquery in your view or _layout page 

<script type="text/javascript" src="Direct2Server.js" ></script>

Passing data to server (Server methods with arguments)

If your server method accepts arguments, like this

public void HiServer(string name, int age)
{
    string receivedName = name;
    int receivedAge = age;
}

Or accepts class object like this

public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }
}

public void HiServer(Person person)
{
    string receivedName = person.Name;
    int receivedAge = person.Age;
}

To pass data to this method, just define a client function which will return the object with arguments values

function serverArguments() {

    var arg = new Object();

    arg.name = "Client";
    arg.age = 20;

    return arg;
}

Now to pass data to server as GET request use attribute getArguments and to pass data as POST request use attribute postArguments and pass this client function reference to them.

So this will make GET request to the server with data

<input type="button" value="Hi Server!" 
         onAjaxServerClick="HomeController.HiServer" 
         getArguments="serverArguments" />

and this will make POST request to the server with data

<input type="button" value="Hi Server!" 
         onAjaxServerClick="HomeController.HiServer" 
         postArguments="serverArguments" />

Receiving server response

Use following attributes to receive server responses

AttributeUse
onServerSuccessResponseCallback function on successfull request
onServerErrorResponseCallback function on error in request

So if your server method returns json data like this

public string HiServer(string client)
{

    // json response
    return "{\"Message\":\"Hi " + client + " \"}";

    // or use any serializer that will convert your object response to json string 
}

Define success callback to receive server response

function successHandler(response) {
    alert('Message received from server - ' + response.Message);
}

And pass it on onServerSuccessResponse event

<input type="button" value="Hi Server!" 
         onAjaxServerClick="HomeController.HiServer" 
         onServerSuccessResponse="successHandler" />

Error response callback has the same signature as jquery ajax error function.

Limitations

It is just a utility for ajax calls so it does have limitations

  1. Works for MVC pattern applications which follows default MVC structure.
  2. Expects server to return data in JSON format.
